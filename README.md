# entry-builder

An es module entry builder.

## entry-builder-config

| 属性 | 描述 | | |
|----|----|----|----|
| entry | 入口 | string \| object | '' |
| entry.path | 入口路径 | string | ''(e.g. './src') |
| output | 输出 | string \| object | '' |
| output.path | 输出路径 | string | ''(e.g. './') |
| output.filename | 输出文件名 | string | 'index' |
